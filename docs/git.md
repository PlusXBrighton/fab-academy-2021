## Useful commands


Pull repo
(git pull include both git fetch and git merge in one handy short cut.) 

> git pull


Add new untracked files to Git

> git add .

Check status of repo:

> git status

Commit  changes

> git commit -m "commit message here"


Push to remote server

> git push
